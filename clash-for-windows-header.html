# 自定义请求头

# 版本要求

0.9.11 版本更新后，支持自定义配置文件下载请求的请求头

# 配置文件

  1. 进入 Settings 界面
  2. 滚动至 Profiles 栏
  3. 点击 Request Headers 右边 Edit 按钮打开编辑器，输入： 
    headers:
  token: "123456"
  cfw-version: "0.9.11"
  4. 点击编辑器右下角按钮保存

保存成功后，更新配置文件（包括自动更新）时，CFW 所发起的请求将会包含上述定义的字段

原文链接：https://clashbk.github.io/clash-for-windows-header.html，转载请注明出处。
