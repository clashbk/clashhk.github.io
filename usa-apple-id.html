美区 Apple ID 账号申请注册教程免费购买

clashgithub 2023-09-21 科学上网

由于苹果 APP Store 的政策原因，部分 APP 在大陆的 APP Store 是被强制下架无法下载的，特别是像科学上网客户端之类的代理 VPN APP，这就需要注册一个美国的 Apple ID 来下载科学上网客户端等大陆 APP Store 无法下载的 APP，本文将详细介绍美国 Apple ID 注册申请教程，无须使用信用卡及科学上网，免费注册美区 Apple ID。

目前来说，获取美国 Apple ID 的方式有以下几种

免费美区 Apple ID

可以在网上搜索免费美区 Apple ID，一般都会有很多，但是这种方式并不是十分安全，特别是有些 APP 需要购买才能使用。

网上购买

这种方式需要花点钱，有点是简单快速，节省时间，一般一个美区 Apple ID 的价格在10元以下，对于追求简单直接的朋友，推荐使用这一方式，但是务必注意购买后更换注册邮箱为自己的邮箱。

自己免费申请美国 Apple ID

通过使用这种方法注册申请美区 Apple ID，无需使用科学上网，也不需要填写信用卡信息，大陆手机号也可以注册，简单三步就可以自己免费申请美国 Apple ID，本文教程注册美国 Apple ID 需要在电脑网页端进行操作。

进入账号创建页面

打开苹果官网，地址：https://www.apple.com/，拉到最底部找到 Manage Your Apple ID，点击进入后点击右上角的 Create Your Apple ID 创建 Apple ID。

或

打开 Apple ID 页面，地址：https://appleid.apple.com/，点击右上角的 Create Your Apple ID 创建 Apple ID。

Apple ID 登录页面

创建 Apple ID 账号

输入基本信息，姓名、国家、生日、邮箱及密码，如电话是大陆手机号则之前的国家选择 China mainland，点击 Continue 注册。

Apple ID 注册页面

修改 Apple ID 信息

打开 Apple ID 页面，地址：https://appleid.apple.com/，登录申请的 Apple ID 账号及密码（如果是刚注册账号可忽略登录步骤）。

Apple ID 个人资料页面

登录后进入个人资料页面，如上图所示，点击 Country / Region 修改 Apple ID 区域为美国，填写美国地址（此步骤为重要步骤），可使用美国地址生成器来获得美国的地址，建议生成美国免税州地址，推荐特拉华州（Delaware） 或蒙大拿州（Montana） 等免税州，电话号码也写美国地址生成器生成的美国电话号码。

由于美国州税率各州不同，建议生成以下五个免税州的地址以减少价格。它们分别是阿拉斯加州（Alaska）、特拉华州（Delaware）、蒙大拿州（Montana）、新罕布什尔州（New Hampshire）和俄勒冈州（Oregon）。

如果是申请其它区域的 Apple ID，原理也是这样。

至此美国 App Store Apple ID 注册申请成功，获得了一个免费的美区 Apple ID，进入 App Store，搜索需要下载的 APP 进行下载安装，如科学上网客户端 Shadowsocks、Quantumult 等。

美区 Apple ID 支付方式

目前美区有三种支付方式，以下简介各种方式的优缺点以及获取成本。

美区 Paypal

  • 要求：美国手机号、美国IP地址、VISA/Mastercard 信用卡（中国发行的即可）

  • 优点：开通iTunes 预授权后可以直接购买，十分方便

  • 缺点：最初注册比较麻烦，容易被风控；且需要有长期可用的美国手机号

礼品卡充值余额

  • 要求：有中国发行的 VISA/Mastercard 信用卡 即可

  • 优点：要求低，有各种官方渠道可以充值

  • 缺点：需要提前充值，到账时间需要10分钟

信用卡

  • 要求：美国银行发行的 VISA/Mastercard 信用卡

  • 缺点：获取成本高

  • 虚拟卡：要么服务商不靠谱，要么手续费高

综上，时间和精力成本上考虑，最简单的是充值方式是购买礼品卡送给自己，收到礼品卡邮件后兑换。

Apple 礼品卡

如果想购买 App Store 中的付费应用或服务，则需要充值礼品卡。

礼品卡获取方式

Apple 礼品卡可以在官网直接购买，也可以在电商平台网站上自行购买，以下分别介绍 Apple 美区官方购买及Amazon 美区购买两种购买平台，其余电商平台包括中国大陆的电商平台也可以购买，但是推荐使用以下两种方式购买 Apple 礼品卡。

Apple 美区官方购买礼品卡

  • 官方链接：点击此处

  • 金额选择：10-200USD 任意金额

  • 优惠：不定期，但较少；可以考虑信用卡返现

  • 到账时间：理论上4小时以内，实际10分钟内

  • 支付方式：国内发行的 VISA/Mastercard 信用卡均可

Amazon 美区购买礼品卡

  • 官方链接：点击此处

  • 金额选择：25-250USD 任意金额

  • 优惠：优惠较多；可以考虑信用卡返现

  • 到账时间：理论上即时，未实践

礼品卡充值方式

打开 App Store，点击右上角头像，点击 Redeem Gift Card or Code，输入礼品卡密码（可选择摄像头扫描或手动输入），最后点击右上角的 Redeem 即可成功充值。

至此你就可以安装任意应用，购买任何付费服务。本文以注册苹果美区账号为例，事实上因为汇率的关系，同样的 App 或服务在不同区可能会存在差价，所以注册外区账号或许也能成为省钱小技巧。
